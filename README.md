# CareerLink – LinkedIn Clone

[**Live Demo → career-link1.vercel.app**](https://career-link1.vercel.app)

---

## Overview

CareerLink is a **LinkedIn clone** built with **Next.js, React, Node.js, Express, and MongoDB**.  
It allows users to:  

- Create and update profiles  
- Connect with other users  
- Post updates, like & comment  
- **Download a resume automatically** generated from profile info  
- Explore connections & network  

**Frontend deployed on Vercel**, **Backend deployed on Render**: [https://careerlink-1-bjdd.onrender.com](https://careerlink-1-bjdd.onrender.com)

---

## Features

- User authentication & profile management  
- Posting, liking & commenting on posts  
- Connections & discover users  
- Dynamic resume generation from profile info  
- Responsive design for desktop and mobile  

---

## Tech Stack

- **Frontend:** Next.js, React, Redux, Axios, CSS Modules  
- **Backend:** Node.js, Express, MongoDB, Mongoose  
- **Deployment:** Vercel (frontend), Render (backend)
